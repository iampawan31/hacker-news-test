# Hacker News App

- Built using ReactJS, TailwindCSS.
- Deployed on Vercel.
- Libraries used include moment.js, loadash, TailwindCSS.

## Installation Steps

- Clone the repo
- In the root directory, execute the following commands

```
npm install
npm run dev
```
